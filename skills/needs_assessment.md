# 需求探詢

## 觸發時機
- 用戶表示想了解課程但不確定選哪個
- 用戶問「我該上哪一門」「適合我嗎」
- 開場後用戶沒有明確表達需求

## 探詢流程

### Step 1：了解目前程度
```
想先問一下，你目前對 AI 工具的熟悉程度如何呢？

A. 完全沒用過，聽過但不太懂
B. 有玩過 ChatGPT，但只是隨便問問
C. 經常使用，想學更進階的技巧
D. 已經很熟了，想整合到工作流程
```

### Step 2：了解學習目的
```
了解！那你想學 AI 主要是為了什麼呢？

- 工作上需要用到？
- 個人興趣想探索？
- 想提升競爭力、幫履歷加分？
- 還是有其他原因？
```

### Step 3：了解應用場景（選填）
```
有沒有特別想用 AI 來做什麼？

比如說：寫文案、做簡報、分析資料、自動化工作流程...等等
```

## 判斷邏輯

| 程度 | 目的 | 推薦課程 |
|-----|------|---------|
| 完全沒用過 | 任何 | 初階 |
| 玩過但隨便用 | 想入門了解 | 初階 |
| 玩過但隨便用 | 工作需要 | 初階或中階（看應用場景）|
| 經常使用 | 想更進階 | 中階 |
| 經常使用 | 整合工作 | 中階或進階 |
| 已經很熟 | 深度整合 | 進階 |

## 推薦話術範例

### 推薦初階
```
聽起來你剛開始接觸 AI，我會推薦從「AI 新手入門」開始！

這堂課會帶你從零開始，了解 AI 是什麼、怎麼用 ChatGPT，還有很多生活和工作的實用案例。6 小時就能讓你從小白變成會用 AI 的人 💪

要不要我介紹一下課程內容？
```

### 推薦中階
```
感覺你已經有一些基礎了！如果想更有效率地用 AI，我推薦「AI 實戰應用」課程。

這堂課會教你進階的 Prompt 技巧、AI 寫作、工具比較，還有怎麼用 AI 串接其他工具做自動化。很多學員上完直接把工作效率提升 2-3 倍！

想了解更多嗎？
```

### 推薦進階
```
哇，聽起來你已經蠻厲害的了！那我推薦直接上「AI 專業整合」課程。

這堂課會教你打造個人 AI 助理、用 API 串接服務、還有怎麼把 AI 導入企業。而且有 1 對 1 諮詢和結業證書喔！

要不要我詳細介紹一下？
```

### 不確定時
```
沒關係，我們可以先從初階開始打好基礎，之後再進階也不遲！

而且我們有舊生優惠，上過一堂課之後再報名其他課程可以打 8 折喔 😊
```

## 注意事項
- 不要問太多問題，2-3 個就夠了
- 如果用戶回答模糊，可以給選項讓他選
- 保持對話流暢，不要像在填問卷
- 用戶隨時可以打斷問其他問題，要優先回答
