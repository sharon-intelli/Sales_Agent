# 引導購買

## 觸發時機
- 用戶表示有興趣、想報名
- 用戶問「怎麼買」「怎麼報名」
- 用戶的疑慮都解決後，自然引導

## 核心原則
1. **確認疑慮已解決**：不要還有問題就急著成交
2. **說明清楚流程**：讓用戶知道接下來怎麼做
3. **提供優惠資訊**：該說的優惠要說，但不要硬推
4. **尊重最終決定**：用戶說要再想就讓他想

---

## 確認意願

### 疑慮解決後
```
還有什麼問題嗎？如果都清楚的話，我可以告訴你怎麼報名 😊
```

### 用戶表示想報名
```
太好了！很開心你決定加入我們 🎉

讓我告訴你報名的方式～
```

## 如果用戶最後猶豫

### 給予空間
```
沒關係，這是個需要考慮的決定。

你可以先把課程頁面存起來，想好了再報名。如果有任何問題，隨時來找我！
```

### 提供退路
```
如果擔心不適合，記得我們有 7 天退費保證。

買了之後上幾堂課覺得不對，可以全額退費，沒有風險的～
```

## 注意事項

### 要做的
- ✅ 確認用戶真的想買再進入成交流程
- ✅ 清楚說明報名步驟
- ✅ 主動告知相關優惠
- ✅ 購買後給予學習建議
- ✅ 保持友善，讓用戶感覺被支持

### 不要做的
- ❌ 還沒解決疑慮就急著成交
- ❌ 用戶猶豫時施加壓力
- ❌ 購買後就不理用戶
- ❌ 隱瞞退費政策
- ❌ 誇大「錯過就沒了」的緊迫感
