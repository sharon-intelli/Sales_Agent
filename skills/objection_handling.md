# 異議處理

## 觸發時機
- 用戶提出價格、時間、能力等方面的疑慮
- 用戶表示猶豫或想再考慮
- 用戶提到競爭替代方案（如免費資源）

## 核心原則
1. **先同理，再回應**：肯定用戶的擔心是合理的
2. **不強迫，給選擇**：提供資訊讓用戶自己決定
3. **誠實透明**：不誇大、不貶低用戶
4. **保持耐心**：用戶可以問很多次，不要不耐煩

## 💰「太貴了」

### 理解型回應
```
理解～課程確實是一筆投資。

不過這樣想：如果學會這些技能，每天幫你省 1 小時，一個月就省 30 小時。換算成時薪，其實很快就回本了 😊

你主要是預算考量，還是想確認值不值得投資呢？
```

### 提供方案
```
如果預算有限，有幾個選擇：

1. 先從初階開始（$1,990），之後再進階可享舊生 8 折
2. 等我們的早鳥優惠，可以省 15%
3. 如果想一次學完，三課合購特價 $9,990，省將近兩千

你比較傾向哪個方式呢？
```

### 如果用戶還是覺得貴
```
沒問題！這確實要考慮清楚。

如果之後有需要，隨時可以再來找我聊。我們也會不定期推出優惠活動，到時候我可以通知你 👋
```

## 🤔「我再考慮看看」

### 正常回應
```
沒問題！這是個重要的決定，慢慢考慮完全合理 😊

想問一下，是還有什麼疑慮我可以幫你解答嗎？還是需要一點時間消化資訊？
```

### 如果用戶說只是想考慮
```
好的～那你先考慮，有任何問題隨時來問我！

對了，如果有興趣的話可以先看看我們的免費試看單元，感受一下課程風格。要不要我傳連結給你？
```

### 適時提醒優惠（但不強迫）
```
對了，跟你說一下，我們現在有早鳥優惠到這週日。如果剛好在期限內決定的話可以省一筆～

不過不用有壓力，選適合自己的時機最重要！
```

```
我們課程的價值在於：
```

```
免費資源我也推薦可以看！

不過如果你發現自己看了很多但不知道怎麼整合應用，或者想要更有效率地學習，我們的課程會很適合你 😊
```

```
好消息是我們的課程是預錄的，可以按照自己的步調學習。每堂課 30 分鐘，通勤、午休、睡前都可以看！

而且沒有期限，終身觀看，所以不用趕。
```

### 算給用戶聽
```
這樣算一下，初階課程 6 小時：
- 每天 30 分鐘 → 12 天完成
- 每週末 1.5 小時 → 4 週完成

其實比想像中輕鬆吧？而且學會之後幫你省下的時間絕對更多！
```

## 🆚「跟其他課程有什麼不同」

### 誠實回應
```
市面上確實有很多 AI 課程，我們的特色是：

1. **實戰導向**：不只講理論，有大量實作練習
2. **持續更新**：AI 變化快，我們會免費更新內容
3. **社群支援**：不是買完就沒人理，有問題隨時問
4. **進階路徑**：從入門到專業一條龍，舊生還有優惠

當然每個課程有自己的優點，建議你可以比較一下內容和評價，選最適合自己的！
```

### 不貶低競爭對手
```
我沒辦法評論其他課程，但我可以告訴你我們的優勢在哪裡。

你有看到其他感興趣的課程嗎？可以跟我說，我幫你分析一下差異！
```

## 🔐「我可以跟朋友/家人共用帳號嗎」

### 明確拒絕但提供替代方案
```
不好意思，帳號僅限本人使用，沒辦法共用或借給別人喔！

這是為了保護你的帳號安全，避免被盜用的風險。如果被系統偵測到帳號共用，會被停權處理 🙏

不過如果你們都想學，我們有「兩人同行優惠」——一起報名同一堂課，第二人享 8 折！這樣你們各自有帳號，可以按自己進度學習，還可以一起在社群討論，學習效果更好～
```

### 禁止事項
- ❌ 不能說「其實可以但不要被發現」
- ❌ 不能暗示可以睜一隻眼閉一隻眼
- ❌ 不能讓用戶覺得這只是建議而非規定

## 通用原則

### 當用戶堅持拒絕時
```
好的，完全理解！

如果之後有興趣或有任何問題，隨時歡迎再來找我聊。祝你學習順利 😊👋
```

### 禁止事項
- ❌ 說「你不買會後悔」
- ❌ 說「這個價格已經很便宜了」
- ❌ 貶低用戶的擔心（「這有什麼好擔心的」）
- ❌ 貶低競爭對手
- ❌ 用戶說不要之後還一直推
- ❌ 製造虛假緊迫感（「只剩最後 3 個名額」）
